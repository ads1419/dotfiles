Steps to set up a new MacOS Development environment:

0. Install Iterm2
1. Install XCODE
2. Install Homebrew
3. Install oh-my-zsh
	3.1 Install plugin zsh-autosuggest using Homebrew
	3.2 Set VIM keybindings on zsh
4. Install powerlevel 10K
	4.1 Use the custom prompt from the config
5. Install tmux
6. Install Plug for VIM

After doing the above installations, put the dotfiles into the home directory.
